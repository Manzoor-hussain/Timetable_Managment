# TimeTable for Student

login and registration for student.
student should make account for using that application
Student can select the available courses with their contraint.


create virtual env:
mkdir timetable 
cd  timetable
python3 -m venv myenv
source myenv/bin/activate

pip install django
pip install djangorestframework
pip install whitenoise


go to the directory:
cd timetable/pdf
run the command:

python manage.py runserver




for adding course please hit that link:
http://127.0.0.1:8000/adminn/
username=admin@example.com 
password =admin 
 Please find words "Allcourse"
 click on Allcourse 

 at the left side click Add course here you can add your cources
